SNS

소개

스프링 프레임워크를 처음 사용하여 진행한 프로젝트입니다. 웹의 기본 소양인 게시판에 여러 기능이 추가된 SNS를 주제로 선정했습니다. 처음에는 프로젝트의 설정을 하는 것조차 미숙했으나, 프로젝트를 통해 프레임워크의 구조와 동작의 흐름을 이해하는데 도움이 되었습니다. 또한 Mybatis를 사용해보며 기본적인 사용법을 숙지할 수 있었습니다.

자세한 내용은 블로그에 있습니다. 블로그로 이동

구현기능

  1. 회원기능
    • 회원가입
    • 로그인
    • 아이디 또는 비밀번호 찾기
    • 회원정보 및 비밀번호 변경
  2. 게시글 기능
    • 게시글 작성, 삭제
    • 타임라인
    • 이미지
    • 검색
  3. 댓글
    • 댓글 작성 및 삭제
    • 댓글에 대한 답글
  4. 좋아요
  5. 프로필/팔로우
    • 프로필화면
    • 프로필수정
    • 팔로우리스트
  6. 실시간 알림
    • 팔로우, 좋아요, 댓글 작성 시 알림 전송
    • 알림 개수 표시


  • 유효성 검사
    • ajax를 이용하여 입력할 때마다 클라이언트에서 확인
    • validator를 이용하여 서버 측에서 검증
    • 아이디는 4~20자로, 영문자와 숫자 구성
    • 비밀번호는 8~30자로, 영문자와 숫자 구성
    • 입력폼의 필수입력란 공백 체크
  • 중복 확인
    • DB에 동일한 아이디 존재 여부 확인
  • Spring security의 BCryptPasswordEncoder 이용한 비밀번호 암호화
  • 로그인
    • DB에 저장된 아이디 및 비밀번호 일치 확인
    • 일치하지 않으면 alert로 메시지를 띄움
    • 일치하면 타임라인(또는 이전 페이지)으로 리다이렉트
  • 인터셉터
    • 비로그인 시 접근 가능한 페이지
      • 회원가입 페이지
      • 로그인 페이지
      • 프로필 페이지
      • 검색 페이지
      • 게시글 상세 페이지
    • 이 외의 페이지에 접근 시 로그인 페이지로 이동
  • 회원정보 변경
    • DB에 저장된 정보는 입력폼에 자동 세팅
    • 현재 비밀번호가 일치하지 않으면 변경 불가
  • 비밀번호 변경
    • 비밀번호 변경과 변경할 비밀번호 일치 여부 확인
    • 현재 비밀번호가 일치하지 않으면 변경 불가
  • 텍스트 입력
    • 최대 300글자까지 작성 가능
    • textarea에 입력된 글자 수 카운트 보여주기
  • 이미지 업로드
    • 다수의 파일 업로드 선택 가능
    • 파일 확장자를 이미지 파일로 제한
    • 업로드할 이미지 파일의 미리보기
  • 텍스트와 이미지 중 하나만 공백이 아니면 게시글 작성 가능
  • 게시글 삭제
    • 자신이 작성한 글에만 ▼ 버튼이 보임
    • 버튼을 클릭하면 팝업창이 뜸
    • 삭제를 클릭하면 다른 팝업창으로 재차 확인
  • 이미지 슬라이드
    • 게시글의 이미지 크기는 동일하게 보여주며, 크기가 클 경우 일부가 잘려서 보임
    • 다수의 이미지는 마우스로 클릭하여 슬라이드로 볼 수 있음
  • 이미지 클릭
    • 브라우저 화면 크기의 팝업창이 뜨며, 가운데에 원본 이미지를 보여줌
    • 이미지 좌우에 있는 버튼을 클릭하면 이미지를 넘겨볼 수 있음
    • 검색
      • 검색할 단어를 입력하면 해당 문자가 포함된 게시글을 보여줌
    • 작성된 댓글은 회원 이름, 아이디, 프로필 사진, 댓글 내용, 날짜 정보 포함
    • 날짜는 초, 분, 시간, 일 단위로 나타내며, 일주일이 지나면 연월일로 보여줌
    • 댓글은 비회원도 볼 수 있으나, 작성은 회원만 가능함
    • 댓글에 대한 답글 작성 기능
    • 삭제된 댓글은 댓글이 삭제되었다는 내용으로 변경
    • 좋아요 기능
      • 흰 하트 버튼을 클릭하면 빨간색 하트로 변경
      • 동시에 옆에 숫자가 올라감
      • 숫자를 클릭하면 좋아요를 클릭한 유저 리스트 팝업창
      • 빨간색 하트를 다시 누르면 좋아요가 취소됨
    • 회원만 가능한 기능으로, 비회원은 인터셉터로 로그인 페이지로 이동
    • 프로필 보기
      • 네비게이션의 프로필 또는 게시글의 회원 이름 클릭 시 이동
      • 회원 이름, 아이디, 프로필 이미지, 소개글, 게시글 수, 팔로우 정보, 작성한 게시글 등을 볼 수 있음
    • 프로필 수정
      • 자신의 프로필 페이지 우측 상단에 있는 버튼을 클릭 시 이동
      • 프로필 이미지와 이름, 소개글 수정 가능
    • 자신의 해당 계정 팔로우 여부를 버튼 배경색으로 구분
    • 팔로우 버튼을 클릭하면 ajax로 팔로잉, 팔로워 수 반영
    • 팔로잉은 회원이 팔로우한 계정, 팔로워는 회원을 팔로우한 계정
    • 팔로우 리스트
      • 팔로잉 또는 팔로워를 클릭하면 리스트 팝업창 뜸
      • 리스트에 이름, 아이디, 프로필 사진과 자신의 해당 계정 팔로우 여부 보여줌
      • 리스트 팝업창에도 팔로우 기능 적용
    • 실시간 알림 전송
      • 팔로우, 좋아요, 댓글 기능 이용 시 대상에게 알림 전송
      • 네비게이션 알림 옆에 읽지 않은 알림 개수 표시
    • 알림 페이지
      • 읽지 않은 알림과 읽은 알림 구분하여 보여줌
      • 팔로우 알림의 경우, 팔로우 여부를 보여줌
      • 댓글 알림의 경우, 댓글 내용을 보여줌
