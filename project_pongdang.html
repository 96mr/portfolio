소개

학부 졸업작품으로 제작했던 게임 소개와 관련 게시글을 작성할 수 있는 웹 사이트입니다.
웹 개발의 기본이라고 생각되는 회원가입, 로그인, 게시판 CRUD를 구현했습니다.
프로젝트를 통해 DB를 연동하고 MVC패턴을 이해하며, 웹 개발의 기본 지식을 쌓을 수 있었습니다.

구현기능

  1. 게시판
    • CRUD 기능
    • 조회수
    • 카테고리 분류 및 페이징 처리
  2. 회원
    • 회원가입
    • 로그인
    • 회원정보조회
    • 작성한 글 보기

  • 게시글리스트
    • 카테고리 별 게시글 분류
    • 10개씩 10페이지 단위로 페이징처리
  • 게시글CRUD
    • 작성 시 로그인 여부 체크
    • 수정, 삭제 시 작성자 일치 확인
    • 작성, 수정 시 게시글의 공백 체크
  • 회원가입 시 gmail SMTP를 이용하여 이메일 인증
  • SHA256으로 비밀번호 암호화
  • 회원정보 조회 및 비밀번호 변겅
  • 내가 작성한 글 모아보기
