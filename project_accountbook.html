가계부

소개

2학년 전공 수업인 MFC 프로그래밍 과제로 만들었던 가계부 프로그램을 개선하여 웹으로 제작했습니다. FullCalender와 Google Chart API를 이용하여 입력된 데이터를 달력과 차트로 볼 수 있습니다.

구현기능

  1. 회원가입 및 로그인
  2. 내역 추가
  3. 캘린더
  4. 차트

  • 유효성 검사
    • ajax를 이용하여 입력할 때마다 확인 가능
    • hibernate validator로 서버 측에서 검증
    • 아이디는 4~20자로, 영문자와 숫자 구성
    • 비밀번호는 8~30자로, 영문자와 숫자 구성
  • 중복확인
    • 입력한 아이디가 DB에 존재 여부 확인
  • BCryptPasswordEncoder 이용한 비밀번호 암호화
  • 특정 페이지에 접근할 때, 인터셉터를 통해 로그인 여부를 확인하여 페이지 이동
  • ajax를 이용하여 지출 또는 수입 라디오 버튼 값에 따라 카테고리 종류(option) 변경
  • 유효성 검사
    • hibernate를 이용하여 서버 측에서 검증
  • fullCalender API 이용
  • ajax를 이용하여 버튼 값에 따라 서버에서 조건에 맞는 데이터를 전달받음
    • 수입, 지출, 잔액 표시
    • 내역은 '카테고리+금액' 형식으로 표시
  • 달력에 나타난 내역을 클릭 시 해당 내역 삭제
  • GoogleChart API 이용
  • ajax를 이용하여 값에 따라 서버에서 조건에 맞는 데이터를 전달받음
    • 수입, 지출, 잔액 표시
    • 수입/지출 라디오 버튼과 달력(연도와 월) 조건
    • 카테고리 별 금액의 비중을 차트에 보여줌
    • 테이블에 내역의 상세리스트를 보여줌
